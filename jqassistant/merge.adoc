== Default Rules

[[merge:MergeDuplicatedJava]]
.Merges duplicated Java nodes and their relationships
[source,cypher,role=concept]
----
MATCH
   (node:Java)
WITH
   node.fqn AS name,
   COLLECT(node) AS nodes,
   COUNT(*) AS count
WHERE
   count > 1
CALL
   apoc.refactor.mergeNodes(nodes, {mergeRels: true}) YIELD node
RETURN
   node
----
